This dashboard provides a detailed view of customer spending habits, coffee sales patterns, and time-based performance insights.
You’ll walkthrough Advance key metrics such as total spend, peak hour spend, cups sold, peak hour cups sold, peak hours, and weekday vs weekend spend. Spending trends all visualized with interactive charts and slicers for deeper analysis.



      📊 Core Analysis Points:

     📌   Caffeine Spending trends

     📌   Spend distribution by weekday and time of day

     📌   Peak hour insights (cups & spend)

     📌   Comparison between weekday and weekend sales
    
     📌   Interactive Button slicers by coffee type
